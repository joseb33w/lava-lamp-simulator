# Lava Lamp Simulator

A full-screen interactive lava lamp toy with animated blobs that rise, cool, merge, split, and drift through a glass lamp. You can change the lava color, liquid color, heat, and blob activity in real time.

## Features

- Full-screen animated lava lamp scene
- Real-time lava and liquid color controls
- Adjustable heat and blob activity
- Blob merging and splitting behavior
- Mobile-friendly single-page experience

## Run

Open `index.html` in a browser.
